This project is to show auto-completion component which is located at /src/AutoCompletion.
File server.js is written to run the project with npm and as mediator for rest api calls (The calls cannot be performed from browser page becase of CORS issue)

To run the program the npm software is to be installed.

To start the program:
1. run "npm install" to install all dependencies.
2. run "npm start". The project is to be available at "http://127.0.0.1:3000"

Any search strings you write will be sent to Wikipedia and use its api.
